# 🏥 숨표 AI: Flutter 디자인 및 사용자 경험 가이드 (Sumpyo AI UI/UX Guide)

**"현대적 모션과 아날로그 감성이 만나는 따뜻한 디지털 약방"**

---

## 🍌 디자인 철학 (Mobile UI Philosophy)

1. **따뜻한 질감과 색채 (Warm & Analog)**:
   - 기계적인 다크 모드나 메탈릭한 컬러를 배제하고, 눈이 편안한 **Warm White**와 **Sage Green**을 사용하여 실제 종이와 자연의 편안함을 제공합니다.
2. **생동감 있는 마이크로 인터랙션 (Emotional Interaction)**:
   - 플로팅(Floating), 바운스(Bouncing) 등 캐릭터와 UI 요소에 생명력을 불어넣어 사용자와 정서적 유대를 형성합니다.
   - Flutter의 고성능 애니메이션(60/120fps)을 활용하여 끊김 없는 피드백을 제공합니다.

3. **고품질 타이포그래피 (Typography for Reading)**:
   - 정보 전달은 Pretendard로 명확하게, 감성 전달은 나눔명조/고운바탕으로 따뜻하게 폰트를 분리하여 사용합니다.

---

## ⚙️ 애니메이션 설계 가이드 (Animation Strategy)

### 1. 상시 애니메이션 (Continuous Motion)

- **캐릭터 플로팅**: 화면에 머무는 동안 약사 캐릭터가 매우 천천히 위아래로 움직임 (Float).
- **주의점**: TickerProvider를 효율적으로 관리하여 배터리 누수를 방지합니다.

### 2. 마이크로 인터랙션 (Micro-interaction)

- **버튼 및 카드 터치**: 탭 하는 순간 살짝 줄어들었다가 튀어오르는 스프링(Spring) 기반 축소/확대 텐션.
- **조제 애니메이션**: 스피너(Spinner) 대신, 캐릭터가 약을 짓는 역동적이고 귀여운 바운스 모션 적용.

### 3. 피드백 제스처 (Feedback)

- **Haptic Feedback**: 결제 완료, 처방전 도착 등 긍정적 경험 시 기분 좋은 진동(HapticFeedback.lightImpact() 등)을 제공.
- **순차적 등장 (Staggered Fade-in)**: 처방전 내용이 한 번에 쏟아지지 않고, 고유번호 → 텍스트 → 추천 음악 순으로 부드럽게 나타납니다.

---

## 🎨 시각적 자산 가이드 (Visual Assets)

- **배경**: 순백색(#FFFFFF) 사용 금지. 항상 #F9F8F6 (Warm White)를 베이스로 사용합니다.
- **텍스트**: 완전한 검은색(#000000) 대신 #4A4A4A (Soft Charcoal)를 사용하여 눈의 피로를 덜어줍니다.
- **카드 및 컴포넌트**: 둥근 모서리(Radius 16~24)와 은은한 난색 계열의 그림자를 사용하여 깊이감을 줍니다.

---

## 🏥 주요 화면 구성 가이드

### 1. 메인 / 고민 접수 (Consultation)

- 부드러운 전구색 조명이 비치는 듯한 배경 렌더링.
- 키보드 활성화 시 전체 레이아웃이 자연스럽게 위로 이동하며 입력에 집중할 수 있는 환경 제공.

### 2. 처방전 화면 (Prescription)

- 화면 가득 차는 '디지털 처방전' 뷰.
- 명조체 텍스트와 세련된 숫자 폰트(Montserrat)의 조화.
- 인스타그램 스토리 등 외부 공유를 위한 9:16 최적화 뷰 모드 제공.
