# 🔍 Research Findings

## Project Concept: Sumpyo AI
- **Core Concept**: Warm Digital Medicine Shop (마음의 쉼표, 따뜻한 디지털 약방).
- **Core Colors**: 
  - #F9F8F6 (Warm White) - Background
  - Sage Green - Primary/Accent
  - #4A4A4A (Soft Charcoal) - Text
- **Typography**: 
  - Pretendard (Functional)
  - Nanum Myeongjo / Goun Batang (Emotional)
  - Montserrat (Numbers)
- **Current UI State**: 
  - Basic `AppBar` with \"마음 약방\" title.
  - Floating message \"어떤 마음의 위로가 필요한가요?\".
  - Basic `TextField` with 16 radius.
  - Custom `SumpyoButton`.

## UI/UX Pro Max Insights
- **Recommended Layout**: Hero > Features > CTA.
- **Recommended Style**: Vibrant & Block-based (modified to Warm & Analog).
- **A11y**: 4.5:1 contrast, 44x44 touch targets.
- **Animation**: Staggered fade-in, spring-based interactions.

## Proposed Improvements
1. **Custom Greeting**: Based on time (e.g., \"좋은 아침이에요, 오늘 기분은 어떠신가요?\").
2. **Glassmorphism/Analog Texture**: Add depth while maintaining warmth.
3. **Mascot Integration**: Use `FloatingMotion` for the Sumpyo character in the Hero area.
